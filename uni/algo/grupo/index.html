function selectionSortIterativo(arr): for i from 0 to n-1: minIdx = i for j from i+1 to n-1: if arr[j] < arr[minIdx]: minIdx = j // nuevo mínimo if minIdx ≠ i: swap(arr[i], arr[minIdx]) // Complejidad: O(n²) siempre // Mejor caso: O(n²) Peor caso: O(n²) // Intercambios: O(n)
Invariante: al terminar la pasada i, los primeros i+1 elementos están ordenados definitivamente.
Comparaciones: n(n-1)/2 — siempre igual.
Ventaja: mínimo de intercambios (a lo sumo n−1).

function selectionSortRecursivo(arr, start): // caso base if start >= n-1: return minIdx = start for j from start+1 to n-1: if arr[j] < arr[minIdx]: minIdx = j if minIdx ≠ start: swap(arr[start], arr[minIdx]) // llamada recursiva selectionSortRecursivo(arr, start+1) // T(n) = T(n-1) + O(n) // Recurrencia → O(n²)
Caso base: start ≥ n−1 (arreglo de 1 elemento).
Llamada recursiva: start+1 (reduce subproblema en 1).
Profundidad de pila: n − igual a la cantidad de pasadas.
Nota: misma complejidad que iterativo, pero usa O(n) de pila.
